𝗏𝐞𝐢𝗅-𝖿𝗋𝐚𝗆𝐞 𝐢𝗌 𝐚 𝗏𝐞𝐢𝗅𝐞𝖽 𝖿𝗅𝐚𝗌𝗄 𝗐𝐞𝖻 𝐚𝗉𝗉 𝖿𝐨𝗋 𝗅𝗌𝖻 𝐞𝗇𝖼𝐨𝖽𝐢𝗇𝗀 (𝗍𝗐𝐢𝗍𝗍𝐞𝗋-𝖼𝐨𝗆𝗉𝐚𝗍𝐢𝖻𝗅𝐞) 𝐚𝗇𝖽 𝖽𝐞𝖼𝐨𝖽𝐢𝗇𝗀 𝗐𝐢𝗍𝗁 𝐚 𝗆𝐮𝗅𝗍𝐢-𝗍𝐨𝐨𝗅 𝐚𝗇𝐚𝗅𝗒𝗓𝐞𝗋 𝗌𝐮𝐢𝗍𝐞. 𝐢𝗍 𝗌𝗁𝐢𝗉𝗌 𝗐𝐢𝗍𝗁 𝖽𝐨𝖼𝗄𝐞𝗋/𝖽𝐞𝗏 𝖼𝐨𝗇𝗍𝐚𝐢𝗇𝐞𝗋 𝗌𝐨 𝐚𝗅𝗅 𝗌𝐭𝗲𝗀𝗈 𝗍𝐨𝐨𝗅𝗌 𝐚𝗋𝗋𝐢𝗏𝐞 𝗉𝗋𝐞𝗂𝗇𝗌𝗍𝐚𝗅𝗅𝐞𝖽.

- 𝐄𝗇𝖼𝐨𝖽𝐞𝗋: `simple_lsb` / `advanced_lsb` (𝗉𝐞𝗋-𝖼𝗁𝐚𝗇𝗇𝐞𝗅) 𝗉𝗅𝐮𝗌 `lsb`, `pvd`, `dct`, `f5`, `spread_spectrum`, `palette`, `chroma`, `png_chunks` 𝗐𝗂𝗍𝗁 𝗉𝐞𝗋-𝗆𝐞𝗍𝐡𝐨𝖽 𝗌𝗎𝖻-𝗈𝗉𝗍𝗂𝗈𝗇𝗌.
- 𝖣𝐞𝖼𝐨𝖽𝐞𝗋: 𝖻𝐢𝗇𝗐𝐚𝗅𝗄, 𝖿𝐨𝗋𝐞𝗆𝐨𝗌𝗍, 𝗌𝗍𝗋𝐢𝗇𝗀𝗌, 𝐞𝗑𝐢𝖿𝗍𝐨𝐨𝗅, 𝗌𝗍𝐞𝗀𝗁𝐢𝖽𝐞, 𝗓𝗌𝗍𝐞𝗀, 𝖻𝐢𝗍-𝗉𝗅𝐚𝗇𝐞 𝖽𝐞𝖼𝐨𝗆𝗉𝐨𝗌𝐢𝗍𝐢𝐨𝗇, 𝐨𝗉𝗍𝐢𝐨𝗇𝐚𝗅 𝐨𝐮𝗍𝗀𝐮𝐞𝗌𝗌 (𝖽𝐞𝐞𝗉 𝗆𝐨𝖽𝐞), 𝗉𝗅𝐮𝗌 𝖻𝐮𝐢𝗅𝗍-𝐢𝗇 𝗌𝐢𝗆𝗉𝗅𝐞 𝖫𝖲𝖡 𝐚𝗇𝖽 𝗌𝐢𝗆𝗉𝗅𝐞 𝗓𝗅𝐢𝖻 𝐞𝗑𝗍𝗋𝐚𝖼𝗍𝐨𝗋𝗌. 𝐀𝗋𝗍𝐢𝖿𝐚𝖼𝗍𝗌 𝐚𝗋𝐞 𝗋𝐞𝗍𝐮𝗋𝗇𝐞𝖽 𝐚𝗌 𝖽𝐚𝗍𝐚 𝐔𝖱𝖫𝗌 𝖿𝐨𝗋 𝐢𝗇𝗅𝐢𝗇𝐞 𝗉𝗋𝐞𝗏𝐢𝐞𝗐/𝖽𝐨𝗐𝗇𝗅𝐨𝐚𝖽.
- 𝐔𝐈: 𝐞𝗇𝖼𝐨𝖽𝐞𝗋/𝖽𝐞𝖼𝐨𝖽𝐞𝗋 𝗍𝐨𝗀𝗀𝗅𝐞, 𝗍𝐨𝐨𝗅𝐢𝗇𝗀 𝗌𝗍𝐚𝗍𝐮𝗌 𝗉𝐚𝗇𝐞𝗅, 𝖻𝐢𝗍-𝗉𝗅𝐚𝗇𝐞 𝗀𝐚𝗅𝗅𝐞𝗋𝗒, 𝖽𝐨𝗐𝗇𝗅𝐨𝐚𝖽𝗌, 𝐚𝗇𝖽 𝗀𝗋𝐚𝖼𝐞𝖿𝐮𝗅 𝗌𝗄𝐢𝗉 𝐨𝖿 𝐨𝐮𝗍𝗀𝐮𝐞𝗌𝗌 𝐢𝖿 𝐢𝗍 𝐢𝗌𝗇'𝗍 𝐢𝗇𝗌𝗍𝐚𝗅𝗅𝐞𝖽.

## 𝖣𝐞𝖼𝐨𝖽𝐞 𝐎𝗉𝗍𝐢𝗈𝗇𝗌
- `Auto Detect` - 𝖿𝐢𝗅𝐞-𝐚𝗐𝐚𝗋𝐞 𝐨𝗋𝖼𝗁𝐞𝗌𝗍𝗋𝐚𝗍𝐢𝐨𝗇 𝐚𝖼𝗋𝐨𝗌𝗌 𝗍𝗁𝐞 𝖽𝐞𝖼𝐨𝖽𝐞 𝗌𝗍𝐚𝖼𝗄, 𝗐𝐢𝗍𝗁 𝗋𝐚𝗇𝗄𝐞𝖽 𝖼𝐚𝗇𝖽𝐢𝖽𝐚𝗍𝐞𝗌.
- `LSB (Least Significant Bit)` - 𝖱𝖦𝖡/𝖱𝖦𝖡𝐀 𝖫𝖲𝖡 𝐞𝗑𝗍𝗋𝐚𝖼𝗍𝐢𝐨𝗇 𝗐𝐢𝗍𝗁 𝗍𝐞𝗑𝗍, 𝗆𝐚𝗀𝐢𝖼-𝖻𝗒𝗍𝐞, 𝐚𝗇𝖽 𝗓𝗅𝐢𝖻 𝐚𝗍𝗍𝐞𝗆𝗉𝗍𝗌.
- `PVD (Pixel Value Differencing)` - 𝗀𝗋𝐚𝗒𝗌𝖼𝐚𝗅𝐞 𝗉𝐚𝐢𝗋-𝖽𝐢𝖿𝖿𝐞𝗋𝐞𝗇𝖼𝐞 𝖽𝐞𝖼𝐨𝖽𝐢𝗇𝗀 𝐮𝗌𝐢𝗇𝗀 𝗌𝗍𝐚𝗇𝖽𝐚𝗋𝖽 𝖯𝖵𝖣 𝗋𝐚𝗇𝗀𝐞𝗌.
- `DCT (Frequency domain)` - 𝖩𝖯𝐄𝖦 𝖣𝖢𝖳 𝖼𝐨𝐞𝖿𝖿𝐢𝖼𝐢𝐞𝗇𝗍 𝗌𝖼𝐚𝗇 𝗐𝐢𝗍𝗁 𝗆𝐢𝖽-𝖻𝐚𝗇𝖽 𝖻𝐢𝗍𝗌𝗍𝗋𝐞𝐚𝗆 𝗋𝐞𝖼𝐨𝗏𝐞𝗋𝗒.
- `F5 (JPEG domain)` - 𝖩𝖯𝐄𝖦 𝗆𝐚𝗍𝗋𝐢𝗑-𝐞𝗇𝖼𝐨𝖽𝐢𝗇𝗀 𝖽𝐞𝖼𝐨𝖽𝐞 𝐨𝗏𝐞𝗋 𝐀𝖢 𝖼𝐨𝐞𝖿𝖿𝐢𝖼𝐢𝐞𝗇𝗍𝗌.
- `Spread Spectrum (Password-based)` - 𝗉𝐚𝗌𝗌𝗐𝐨𝗋𝖽-𝗌𝐞𝐞𝖽𝐞𝖽 𝖼𝐨𝗋𝗋𝐞𝗅𝐚𝗍𝐢𝐨𝗇 𝐞𝗑𝗍𝗋𝐚𝖼𝗍𝐢𝐨𝗇 𝗐𝐢𝗍𝗁 𝖼𝐨𝗇𝖿𝐢𝖽𝐞𝗇𝖼𝐞 𝗌𝗍𝐚𝗍𝗌.
- `Palette (Color index encoding)` - 𝐢𝗇𝖽𝐞𝗑𝐞𝖽 𝗉𝐚𝗅𝐞𝗍𝗍𝐞 𝗌𝗍𝐫𝐞𝐚𝗆 𝐞𝗑𝗍𝗋𝐚𝖼𝗍𝐢𝐨𝗇 𝖿𝐨𝗋 𝖯𝖭𝖦/𝖦𝐈𝖥.
- `Chroma (Color channel hiding)` - 𝖸𝖢𝖻𝖢𝗋 𝖼𝗁𝗋𝐨𝗆𝐚-𝖼𝗁𝐚𝗇𝗇𝐞𝗅 𝖻𝐢𝗍𝗌𝗍𝗋𝐞𝐚𝗆 𝐞𝗑𝗍𝗋𝐚𝖼𝗍𝐢𝐨𝗇.
- `PNG Chunks (Metadata)` - 𝗍𝐄𝖷𝗍/𝐢𝖳𝖷𝗍/𝗓𝖳𝖷𝗍 𝗉𝐚𝗋𝗌𝐢𝗇𝗀 𝗐𝗂𝗍𝗁 𝖽𝐞𝖼𝐨𝗆𝗉𝗋𝐞𝗌𝗌𝐢𝗈𝗇 𝐚𝗇𝖽 𝖼𝗁𝐮𝗇𝗄 𝐢𝗇𝗏𝐞𝗇𝗍𝐨𝗋𝗒.

## 𝖫𝐨𝖼𝐚𝗅 𝗌𝐞𝗍𝐮𝗉/𝗋𝐮𝗇 (𝗇𝐨 𝖣𝐨𝖼𝗄𝐞𝗋)
